Приложение реализует парсинг json файла.

Объект класса MyJSONParser реализует считывание данных из файла, расположенного по адресу
<filePath>/<fileName> (по умолчанию: файл test.json в корневом каталоге).

Объект класса MyJSONParser реализует парсинг файла (реализуется по известным полям json файла). Содержит переменную name типа String и коллекцию people типа People для хранения данных второго уровня. Доступ к данным осуществляется по соответствующим методам set...()


По нажатию кнопки "Показать" на экран выводятся все значения из json файла. CheckBox служит для вывода данных по определенному id, который задается в EditText (по умолчанию равен 0).
